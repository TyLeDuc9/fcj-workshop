* Get connected and become familiar with the members of the First Cloud Journey program.
* Understand the basic AWS services and create an AWS account to complete the assigned tasks.

### Week 1 Achievements:

* Gained a fundamental understanding of AWS and its core service categories.
  * Completed the assigned learning tasks and modules.
  * Successfully created IAM users and administrator groups.
* Successfully created and configured an AWS Free Tier account.
* Successfully created and configured:
  * Virtual Private Cloud (VPC)
